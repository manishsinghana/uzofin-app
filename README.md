# React + TypeScript + Vite Project

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.


## Getting Started

### 1. Install Dependencies

Use `npm` to install the project dependencie:

```bash
npm install
```

### 2. Run Development Server

Start the development server:

```bash
npm run dev
```
Then open [http://localhost:5173] in your browser to see the result.

### 4. Build Before Pushing

Before pushing your changes, build the project and `make sure to resolve all errors and warnings`:

```bash
npm run build
```
This will generate the optimized production build in the `dist` directory.
